# Auditory Decoding Paradigms

This repo captures a legacy Psychtoolbox (Octave) paradigm, extracts its behavioral contract, and supports a clean rewrite in PsychoPy 2024.2.4 on Windows 10.

## Structure
- legacy_paradigm/: placeholder for the original Psychtoolbox/Octave code
- docs/: contracts, intake notes, and decisions
- psychopy/: target rewrite in PsychoPy
